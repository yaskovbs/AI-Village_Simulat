# סימולטור כפר AI

סימולטור כפר AI הוא כלי ניהול מקיף למפתחות API של ספקי בינה מלאכותית מובילים המשמשים להפעלת סימולציית כפר עם סוכני AI. האפליקציה מספקת ממשק נוח לניהול מפתחות API, מעקב אחרי השימוש, וויזואליזציה של נתוני צריכה בסימולציית הכפר.

## ✨ תכונות עיקריות

- **ניהול מפתחות API**: תמיכה ב-OpenAI, Anthropic (Claude), Google (Gemini), YouTube Data API v3 וחיפוש באינטרנט
- **לוח בקרה אינטראקטיבי**: תצוגה ויזואלית של שימוש ב-API לאורך זמן
- **גרפים ותרשימים**: ויזואליזציה של נתוני השימוש עם תרשימים אינטראקטיביים
- **סימולציית כפר**: מערכת סוכני AI שמתנהגים בכפר וירטואלי
- **עיצוב רספונסיבי**: תמיכה מלאה במכשירים ניידים ומחשבים שולחניים
- **תמיכה בעברית**: ממשק מותאם לכיוון כתיבה מימין לשמאל (RTL)
- **ערכות נושא**: תמיכה בערכות נושא בהירה וכהה
- **אחסון מקומי**: שמירת מפתחות API באופן מאובטח בדפדפן

## 🚀 התחלה מהירה

### דרישות מקדימות

- Node.js (גרסה 16 ומעלה)
- npm (מגיע עם Node.js)

### התקנה

1. שכפל את המאגר:
```bash
git clone https://github.com/yaskovbs/AI-Village_Simulat.git
cd AI-Village_Simulat
```

2. התקן את התלויות:
```bash
npm install
```

3. הפעל את שרת הפיתוח:
```bash
npm run dev
```

4. פתח את הדפדפן בכתובת `http://localhost:5173`

## 📖 שימוש

### ניהול מפתחות API

1. עבור ללשונית "ניהול מפתחות API"
2. לחץ על כרטיס הספק הרצוי (OpenAI, Anthropic, Google, YouTube, או חיפוש)
3. הזן את מפתח ה-API שלך
4. לחץ על "שמור" כדי לשמור את המפתח באופן מקומי

### צפייה בנתוני השימוש

1. עבור ללשונית "לוח בקרה ושימוש"
2. צפה בגרף השימוש השבועי
3. הנתונים מוצגים לפי ספק ויום

## 🏘️ אודות סימולציית הכפר

סימולטור הכפר AI כולל:
- **מפת עולם**: רשת עם תאים של דשא, מים ובתים
- **סוכני AI**: ישויות חכמות שמתנהגות באופן אוטונומי
- **התנהגויות**: הסוכנים יכולים לנוע, לתקשר, ולבצע משימות
- **שימוש ב-API**: הסוכנים משתמשים במפתחות ה-API השמורים להתנהגות חכמה

## ⚠️ אזהרות אבטחה

- מפתחות ה-API נשמרים באופן מקומי בדפדפן בלבד
- **אין להשתמש בפתרון זה בסביבת ייצור**
- מומלץ להשתמש בפתרונות אחסון מאובטחים יותר עבור סביבות ייצור

## 🛠️ טכנולוגיות

הפרויקט בנוי עם הטכנולוגיות המודרניות ביותר:

- **React 18** עם Hooks
- **TypeScript** לבדיקת טיפוסים סטטית
- **Vite** כסביבת פיתוח ובנייה מהירה
- **Shadcn/ui** כספריית רכיבי ממשק משתמש
- **Tailwind CSS** לעיצוב מהיר ומותאם אישית
- **next-themes** לניהול ערכות נושא
- **D3.js** לויזואליזציה של נתונים
- **ESLint** לשמירה על איכות קוד
